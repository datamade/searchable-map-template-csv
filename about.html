About

This is my second iteration of a map to use my notes on places to eat to make it easier to find somewhere to eat. I originally use Awesome Table, but their pricing pushed me to find an alternative given that this is just a personal project. Thus, after finding searchable-map-template-csv by Derek Eder and DataMade, I figured it was time to shake off some javascript and html rust to make a second iteration of a food map!



Privacy Notice

One note is that your general location is found using an API from ipwhois.io. This allows the program to start the map at a location close to where you are rather than a default. The data is only held in this specific window and is not recorded anywhere.
